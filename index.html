Building GenetICS

View on GitHub

Follow instructions

Follow instructions on initializing a build environment up to a point of downloading the source code.

Download CM9 source code

$ repo init -u git://github.com/GenetICS/manifest.git -b ics
$ repo sync

Update ROM Manager & Terminal Emulator. (You have to do this again only when the apps have been updated.)

$ cd CM9_WORKING_DIR/vendor/cm/
$ ./get-prebuilts

Apply patches

To install

$ cd CM9_WORKING_DIR/
$ sh device/lge/msm/msm7x27-common/patches/install.sh

To uninstall

$ cd CM9_WORKING_DIR/
$ sh device/lge/msm/msm7x27-common/patches/uninstall.sh

Compile the ROM

$ cd CM9_WORKING_DIR/
$ sh build/envsetup.sh && lunch cm_p500-userdebug
$ make bacon -j4

Flash ZIP

out/target/product/DEVICE/cm-VERSION-DEVICE.zip

Use 'repo sync' to pull merged changes from upstream. Use 'make clobber' to clean the build environment. Repeat the last step to do a new build.
